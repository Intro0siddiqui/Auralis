Features

📥 Media Download

Download audio from YouTube videos and Instagram Reels in MP3, FLAC, AAC, OGG, WAV, or M4A format.

🎵 Music Player

Full-featured player with queue, repeat, shuffle, and background playback.

📂 Local Import

Import music from any folder on your device. Scan system directories automatically.

🔄 P2P Sync

Sync your library between devices over LAN. QR code or temp code pairing.

🏷️ Metadata

Edit track info, album art, and organize with playlists.

⚡ Lightweight

Low memory usage, native performance. No Electron, no bloat.

Build from Source

git clone https://github.com/Intro0siddiqui/Auralis.git
cd Auralis/auralis
./gradlew :desktop:run              # Run desktop app
./gradlew :desktop:packageDmg       # Build macOS .dmg
./gradlew :desktop:packageDeb       # Build Linux .deb
./gradlew :desktop:packageRpm       # Build Linux .rpm
./gradlew :desktop:packageAppImage  # Build portable Linux binary
./gradlew :android:assembleDebug    # Build Android APK
